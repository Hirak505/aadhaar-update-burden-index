# aadhaar-update-burden-index

Data-driven analysis of UIDAI Aadhaar enrolment and update data to compute the Aadhaar Update Burden &amp; Inclusion Index (AUBII) and generate actionable governance insights.

# ‚Äì Admin Dashboard

This repository contains the Streamlit-based visualization layer for
** AI-Driven Operational Integrity & Anomaly Detection
for the Aadhaar Ecosystem**.

The dashboard enables administrators to visually identify:

- Normal districts
- Hot Spot districts (abnormally high activity)
- Dark Spot districts (abnormally low activity)
- Risk severity levels
- Z-score anomaly intensity
- State-wise anomaly density
- Top red-flagged districts

- State-wise anomaly density heatmap
- Top red-flagged district table
